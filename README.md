# Find drugstore open

This is an assessment app that shows a drugstore based on arbitrary criteria.

## Requirements
 - docker
 - docker-compose
 - git

## Run

Clone it and run it

```
git clone https://github.com/j1cs/drugstore.git
docker-compose up -d
```

Get you public ip
```
curl https://ipinfo.io/ip
x.x.x.x
```

Open the result (x.x.x.x) in your browser.
